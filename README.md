# deciduously_com

Minimal template for created a new [Sunfish](https://github.com/tangramdotdev/sunfish/)/[Pinwheel](https://github.com/tangramdotdev/pinwheel/) website.

## Usage

```
$ HOST=0.0.0.0 PORT=8080 cargo run --release
```
